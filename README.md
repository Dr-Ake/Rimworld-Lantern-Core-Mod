# LanternsCore (RimWorld 1.6)

LanternsCore is a framework mod for the Emotional Spectrum / Lantern rings.
It provides generic, lore-friendly Lantern abilities (constructs, summons, barriers, heals, auras, blasts), charge handling, auto-selection/delivery, batteries/recharge, costumes, and flight.

**Add-on authors:** start here:
- `Docs/HowTo_CreateLanterns.md` — complete no-C# guide + full template.
- `Docs/AddonHooks.md` — optional advanced hooks.
- `Docs/Compatibility.md` — what is safe to depend on.
- `Docs/SmokeTest.md` — quick manual verification steps.
- `CHANGELOG.md` — release notes.

**Example add-on:** see `Docs/Example_Addon_YellowLantern.xml`.
