# LanternsCore (RimWorld 1.6)

LanternsCore is a framework mod for Emotional Spectrum Lantern rings and "hero gear" style mods (rings, belts, suits, artifacts, etc.).
It provides generic abilities (constructs, summons, barriers, heals, auras, blasts), charge/resource handling, auto-selection/delivery, batteries/recharge, costume transformation, and flight.

**Add-on authors:** start here:
- `Docs/HowTo_CreateLanterns.md` - complete no-C# guide + full template.
- `Docs/AddonHooks.md` - optional advanced hooks.
- `Docs/Compatibility.md` - what is safe to depend on.
- `Docs/SmokeTest.md` - quick manual verification steps.
- `Docs/CommunityTestPlan.md` - checklist for playtesters/bug reports.
- `Docs/BugReportTemplate.md` - copy/paste issue template.
- `LanternsCore_ModBuilder/Guide.md` - excruciatingly detailed Builder guide.
- `CHANGELOG.md` - release notes.
